# Giako's dotfiles

Dotfiles I use for different programs:

- vim
- tmux
- oh-my-zsh

Dotfiles are versioned using [git](https://git-scm.com/) and configured via [chezmoi](https://chezmoi.io).

# How to install

```shell
chezmoi init https://github.com/Giako/dotfiles.git
chezmoi apply
chezmoi update
```
